HappyTail is a responsive single-page application built with Vue 3 and Vite. The UI was designed to be simple and friendly, with support for dark mode, variable text sizing, and a clean page structure. This project was part of SIT120 coursework and served as a hands-on exercise in modern frontend development using component-based architecture.

## Features

- Responsive layout (desktop & mobile)
- Dark mode toggle
- Adjustable text sizes (SM / MD / LG)
- Interactive UI components using Bootstrap 5
- Page navigation via Vue Router
- State management using Pinia
- Lightweight and fast thanks to Vite dev environment

## Tech Stack

- Vue 3
- Vite
- Bootstrap 5
- Pinia (state store)
- JavaScript
- HTML / CSS
